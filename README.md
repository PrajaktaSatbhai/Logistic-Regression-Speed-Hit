# Logistic Regression: Speed vs Hit

This project demonstrates the implementation of Logistic Regression using
both **scikit-learn** and **statsmodels**.

## Dataset
- **Speed**: Numeric feature
- **Hit**: Binary target variable (0 / 1)

## Methods Used
- Logistic Regression (sklearn) for prediction
- Logistic Regression (statsmodels) for statistical inference

## Libraries
- numpy
- pandas
- scikit-learn
- statsmodels

## Output
- Classification report
- Model coefficients
- p-values and confidence intervals
